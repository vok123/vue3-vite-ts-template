# Installing VSCode Plugins

- Volar
- Prettier
- UnoCSS
- vscode-styled-components
- ESLint
- Code Spell Checker
- Auto Close Tag
- Auto Rename Tag
- Google Translate
- Path Autocomplete
- GitLens

# Getting Started

- dev
```bash 
npm i
npm run dev
```

- test
```bash 
npm i
npm run test
```


- build
```bash 
npm i
npm run build
```
